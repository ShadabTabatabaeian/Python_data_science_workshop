Introduction to Data Science with Python

Welcome! This workshop is an introduction to data science using Python and requires no prior experience.

What We Learn

  • Start with the basics of Python programming and move into the foundations of machine learning.
  • Explore what data science is, why it matters, and our ethical responsibilities as data scientists.
  • Build technical skills plus the critical thinking and care needed to use data for good.

Goals and Outcomes

  • Gain a solid foundation in Python and simple statistical modeling.
  • Become familiar with a range of machine learning methods and algorithms.
  • Learn which Python tools to use and what questions each method can answer.
  • Develop the judgment to choose the best approach for your own data questions.
