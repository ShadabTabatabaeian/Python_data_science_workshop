# Intro to Data Science in Python (Undergraduate)

A 6-session introductory course with two notebooks per session:
- `topics.ipynb`: concept tutorial with explanations and examples
- `challenge.ipynb`: data challenge based on the session topics

## Sessions
1. Python Foundations
2. Data Wrangling with pandas
3. Data Visualization
4. Statistics & Probability Basics
5. Intro to Machine Learning
6. Capstone Mini-Project

## How to Use
- Open notebooks in Jupyter or VS Code.
- Work through `topics.ipynb` first, then complete `challenge.ipynb`.

## Repo Layout
- `sessions/<session-id>/topics.ipynb`
- `sessions/<session-id>/challenge.ipynb`
- `data/` for datasets used in challenges
- `assets/` for images and course visuals
