# Syllabus (Draft)

**Format:** 6 sessions, 2–3 hours each. Each session includes a tutorial notebook and a data challenge notebook.

## Session 1 — Python Foundations
- Data types: numbers, strings, printing, lists, dictionaries, booleans, tuples, sets
- Comparison operators
- `if`, `elif`, `else`
- `for` loops, `while` loops, `range()`
- List comprehension
- Functions, lambda expressions
- `map` and `filter`
- Methods

## Session 2 — Data Wrangling with pandas
- DataFrames, Series, reading CSVs
- Filtering, selecting, sorting
- Groupby and aggregation
- Missing values

## Session 3 — Data Visualization
- Matplotlib basics
- Seaborn charts
- Best practices in labeling

## Session 4 — Statistics & Probability Basics
- Descriptive statistics
- Distributions
- Sampling
- Correlation vs causation

## Session 5 — Intro to Machine Learning
- Supervised vs unsupervised
- Train/test split
- Linear regression
- Classification (k-NN or logistic regression)

## Session 6 — Capstone Mini-Project
- Problem framing
- Data cleaning + EDA
- Basic model + evaluation
- Short write-up
